【競プロ】累積和について学ぼう！

March 30, 2023

はじめに

競技プログラミングの世界で有名なテクニックの一つとして累積和があります。
累積和とは簡単に言うと、

前処理を行って、区間の総和のクエリに対する結果を$O(1)$で取得できる手法

です。
今回の記事では実際のAtCoderの問題を解くことによって累積和をどのように使えばいいのか見ていきたいと思います。

問題1:Lottery

URL

https://atcoder.jp/contests/tessoku-book/tasks/tessoku_book_ce

問題文

太郎君はくじを $N$ 回引き，$i$回目の結果は $A_i$ でした．$A_i=1$ のときアタリ，$A_i=0$ のときハズレを意味ます． 「 $L$ 回目から $R$ 回目までの中では，アタリとハズレどちらが多いか？」という形式の質問が $Q$ 個与えられるので， それぞれの質問に答えるプログラムを作成してください

解説

$S_i:=A$ の $[0,i)$ の総和 として $Q$ 個のクエリにたいして $S_r-S_{l-1}$ をチェックすればよい。
以下は実装例である。

#include <bits/stdc++.h>
using namespace std;

int N, Q;
int A[100010];
int L[100010], R[100010];
int S[100010];

int main() {
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        cin >> L[i] >> R[i];
        L[i]--;
    }
    for (int i = 0; i < N; i++) {
        S[i + 1] = S[i] + A[i];
    }
    for (int i = 0; i < Q; i++) {
        int x = S[R[i]] - S[L[i]];
        int y = R[i] - L[i] - x;
        if (x == y) {
            cout << "draw" << endl;
        } else if (x < y) {
            cout << "lose" << endl;
        } else {
            cout << "win" << endl;
        }
    }
    return 0;
}
